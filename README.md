# tempest

Téléchargement et initialisation du projet
```
git clone --recurse-submodule git@git.unistra.fr:thery/sdl2_base_vectors.git
mkdir BuildSDLEx
cd BuildSDLEx
cmake ../sdl2_base_vectors
make -j 8
```

## Contrôle du jeu

D -> aller à droite
Q -> aller à gauche
F -> activer la bombe
Espace -> tirer
